## 遇到的问题

- 当前系统 `python` 命令是 Windows Store 占位入口，不能直接运行 Python
- Codex bundled Python 可用，已用它创建 `.venv`
- 端口 8000 在当前环境中会返回代理层 502，后端在 8010 端口验证通过
- Python `httpx` 默认读取环境代理变量，请求 localhost 时需使用 `trust_env=False`
- `.env` 暂无 `ZHIPU_API_KEY`，因此本次 M4b 验收使用本地 embedding fallback；接入真实 key 后需重跑 seed
- 本地 fallback 只用于开发验收，语义效果弱于真实 embedding；当前已切换到真实 embedding 索引，后续 Agent 推荐质量以真实索引为准
