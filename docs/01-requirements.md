# 项目需求说明

## 项目定位

AI Shopping Guide 是一个基于 RAG 的多模态电商智能导购 Web Demo。项目用正常 Web 前端替代原生 App，面向课程/答辩演示，重点展示从内容浏览到购买决策再到模拟下单的完整链路。

## 核心目标

- 支持用户通过文本、图片、语音与导购 Agent 交互。
- 基于商品知识库进行 RAG 检索，推荐真实存在的商品。
- 在对话流中展示商品卡片，并支持加入购物车。
- 支持自然语言购物车管理和模拟下单。
- 保持本地可运行、云端可部署、文档可解释。

## 功能范围

### 必做

- 手机号/邮箱简单登录。
- 导购聊天窗口。
- 文本输入与 SSE 流式回复。
- 商品卡片展示。
- 商品详情、SKU、价格、图片展示。
- RAG 商品检索。
- 购物车加购、删除、改数量。
- 模拟订单确认。
- 历史会话查看与管理。

### 加分

- 图片找货。
- 语音输入。
- TTS 回复播报。
- 多轮上下文记忆。
- 排除条件解析。
- 多商品对比。

## 商品范围

首版支持以下 4 个类目商品：

| 类目 | 二级类目示例 | 商品数量 |
|------|------------|---------|
| 美妆护肤 | 精华、面霜、洁面、防晒、面膜等 | 10+ |
| 数码电子 | 智能手机、平板电脑、耳机、智能手表等 | 10+ |
| 服饰运动 | 短袖T恤、运动鞋、冲锋衣、瑜伽裤等 | 10+ |
| 食品饮料 | 坚果零食、咖啡、功能饮料、方便食品等 | 10+ |

数据来源为项目内 `data/` 目录下的结构化 JSON 商品数据，不依赖外部爬虫或第三方 API。

## 可靠性要求

- 不编造商品、价格、优惠、库存、SKU。
- 关键商品字段必须来自结构化数据。
- 检索不到合适商品时，需要明确说明并建议用户补充条件。
